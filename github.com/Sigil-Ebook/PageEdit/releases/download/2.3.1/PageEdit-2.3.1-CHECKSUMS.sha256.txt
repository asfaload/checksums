Copy the downloaded file to your Desktop.2. Navigate to /Applications/Utilities and double click on Terminal.app3. Enter the following lines in Terminal.app hitting return after each one    cd    cd Desktop    shasum -a 256 PageEdit.app-2.3.1-Mac-x86_64.txz4. The resulting shasum should match **exactly** the same string for that file in PageEdit-2.3.1-CHECKSUMS.sha256.txt5. To leave Terminal.app enter the following:    exit**How to Verify SHA 256 Checksums on Windows**Borrowed and Modified from Apache OpenOffice:     https://www.openoffice.org/download/checksums.html#hash_winThere are several tools available:    "FSUM" - http://www.slavasoft.com/fsum/    "Quick Hash GUI" - http://sourceforge.net/projects/quickhash/    "SHA256 Checksum Utility" - https://kanguru.zendesk.com/entries/21747773-SHA256-Checksum-Utility    "File Checksum Tool" - http://www.krylack.com/file-checksum-tool/ The following instructions are for the "File Checksum Tool":  1.
